# Avalanche OS — Asset Pack

Backcountry splitboard-touring Linux distro. Pre-dawn alpine palette.

## Logo / icon (vector — scale freely)
- `avalanche-mark.svg` — primary two-tone mark, for DARK backgrounds
- `avalanche-mark-onlight.svg` — two-tone mark, for LIGHT backgrounds
- `avalanche-mark-mono-white.svg` — single-color silhouette (white)
- `avalanche-mark-mono-beacon.svg` — single-color silhouette (orange)
- `avalanche-wordmark-horizontal.svg` — icon + "Avalanche OS" lockup (uses Space Grotesk; install the font or open in a browser with internet for correct type)

## Icon (raster PNG, transparent)
- `icon/avalanche-icon-512.png` · `-256` · `-128` · `-64` — two-tone app icons
- `icon/avalanche-icon-onlight-512.png` — for light surfaces
- `avalanche-mark-mono-white-512.png` — mono silhouette

- `avalanche-wallpaper-dark-3840x2160.png` — with subtle corner mark + planted splitboard

**Light — overcast high-alpine whiteout**

## Boot splash
- `avalanche-bootsplash-1920x1080.png` — static white mark on Crevasse #07121E
- `bootsplash/avalanche-boot-1.png` → `-2` → `-3` — 3-stage progression backgrounds (1920×1080). Each frame lights one more peak: summit → slab → beacon, giving a loading impression. Use as sequential Plymouth backgrounds (hold each for ~0.7–0.9s, advance on progress, land on frame 3 at handoff).
- For a redrawn Plymouth animation use `avalanche-mark-mono-white.svg` as the source silhouette.
